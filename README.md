
# Activo

Simple web application for saving your training sessions and sharing them with other users.


## Run Locally

Clone the project

```bash
  git clone https://github.com/igorw33/activo.git
```

Go to the project directory

```bash
  cd [project directory]
```

Compile the project and run App

Open localhost on port 4567 in web browser of your choice

## Features

- Add, edit and delete your activities
- Main page where it's possible to see other users activities
- Like button
- Set goals for your activities
- User registration system
- Admin page

Note: Backend was not a priority for this project, so due to time limitations when writing this project only one user can be logged at a time.  
Important: Passwords and usernames are stored in local database in plaintext.

## Tech Stack

**Client:** JavaScript, HTML, CSS

**Server:** Java version 23.0.1, Spark, SQLite
